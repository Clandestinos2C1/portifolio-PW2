# 💻 Programação Web 2 — Portfólio de Projetos (PW2)

Este repositório reúne as atividades e projetos desenvolvidos na disciplina **Programação Web 2**, com foco na construção de aplicações web dinâmicas utilizando **PHP** e integração com banco de dados.

A proposta é concentrar em um único lugar os exercícios feitos em aula, testes de funcionalidades e pequenos sistemas criados durante o aprendizado da matéria.

Ao longo deste repositório, é possível acompanhar a evolução desde scripts PHP simples até aplicações com estrutura mais organizada e conectadas ao banco de dados.

---

## 📌 Finalidade do Repositório

Este portfólio foi criado para:

* Registrar atividades desenvolvidas em aula
* Praticar programação com PHP
* Desenvolver aplicações web dinâmicas
* Trabalhar com formulários e processamento de dados
* Integrar PHP com banco de dados
* Organizar os projetos da disciplina PW2
